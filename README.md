# ⌨️ Тренажёр слепой печати

Приложение для тренировки скорости и точности набора текста. Подходит для изучения слепой печати на русской и английской раскладке.

---

## 👤 Для пользователей — как начать

### Вариант 1: Готовый exe (самый простой)

1. **Скачайте** файл `TypingTrainer.exe` (из [Releases](../../releases) или получите от автора)
2. **Запустите** — дважды кликните по файлу
3. Готово! Установка не нужна.

Файлы `progress.json`, `settings.json` и папка `assets` создадутся автоматически рядом с exe.

### Вариант 2: Запуск из исходников (если есть Python)

1. Скачайте или клонируйте репозиторий
2. Дважды кликните **`run.bat`** — он сам установит зависимости и запустит приложение

Или вручную:
```bash
pip install -r requirements.txt
python main.py
```

## 📖 Как пользоваться приложением

| Действие | Как сделать |
|----------|-------------|
| Выбрать язык | RU или EN вверху экрана |
| Выбрать режим | Нажмите на кнопку: Фразы, Слова, Буквы, Цифры и т.д. |
| Начать печать | Начните вводить текст в поле ввода |
| Изменить размер шрифта | Кнопки **−** и **+** в экране тренажёра |
| Вернуться в меню | Кнопка **← Назад в меню** |
| Посмотреть прогресс | Кнопка **Прогресс** в шапке |
| Посмотреть статистику | Кнопка **Статистика** (графики по дням/месяцам/годам) |

**Подсказка:** зелёный текст — правильно, красный — ошибка, жёлтый — текущий символ.

## 🛠 Для разработчиков

### Возможности

- Режимы: фразы, слова, буквы, цифры, скороговорки, цитаты, код, смешанный
- Языки: RU и EN
- Прогресс и статистика с графиками (matplotlib)
- Регулировка размера шрифта (сохраняется между сессиями)
- История упражнений в `progress.json`

### Сборка exe

```bash
build.bat
```

Результат: `dist/TypingTrainer.exe`

### Сборка архива для раздачи

```bash
distribute.bat
```

Создаёт `dist/TypingTrainer.zip` с exe и инструкцией «Как пользоваться.txt».

### Создание установщика (Inno Setup)

1. Установите [Inno Setup](https://jrsoftware.org/isinfo.php)
2. Соберите exe: `build.bat`
3. Откройте `installer.iss` в Inno Setup → Compile

Или из командной строки:
```bash
"C:\Program Files (x86)\Inno Setup 6\ISCC.exe" installer.iss
```

Результат: `Output/TypingTrainerSetup.exe` — установщик с ярлыками в меню Пуск и на рабочем столе.

### Docker

**Сборка образа:**
```bash
docker build -t typing-trainer .
```

**Запуск (с Xvfb, для тестов):**
```bash
docker run --rm typing-trainer
```

Или через Compose:
```bash
docker-compose build
docker-compose up
```

**Запуск с отображением GUI на Linux (X11):**
```bash
xhost +local:docker
docker run --rm -e DISPLAY=$DISPLAY \
  -v /tmp/.X11-unix:/tmp/.X11-unix \
  --entrypoint python typing-trainer main.py
```

> На Windows для GUI удобнее использовать `TypingTrainer.exe` или `run.bat`.
