# Тренажёр слепой печати

Приложение для тренировки скорости и точности набора текста.

## Возможности

- **Интерфейс тренировки** — печатайте фразы, видите в реальном времени правильность ввода (зелёный — верно, красный — ошибка)
- **Скорость печати** — отображается в символах в минуту (зн/мин)
- **Прогресс** — окно «Прогресс» показывает историю: дата, время, скорость по каждой сессии и среднюю скорость
- **Режимы упражнений** — фразы, частые слова, случайные буквы и цифры; выбор через выпадающее меню в верхней панели

## Запуск

```bash
pip install -r requirements.txt
python main.py
```

## Сборка в exe (PyInstaller)

```bash
# Windows
build.bat
```

Готовый файл: `dist/TypingTrainer.exe`

## Установщик для других пользователей (Inno Setup)

1. Установи [Inno Setup](https://jrsoftware.org/isinfo.php).
2. Убедись, что собран файл `dist/TypingTrainer.exe` (см. раздел выше).
3. Открой `installer.iss` в Inno Setup и нажми **Build** (или запусти из терминала):

```bash
cd typing_trainer
"C:\Program Files (x86)\Inno Setup 6\ISCC.exe" installer.iss
```

После сборки появится установщик: `Output/TypingTrainerSetup.exe`.

### Как другим пользователям установить

- Отправь файл `TypingTrainerSetup.exe` (через архив, диск, мессенджер).
- Пользователь запускает его, проходит установку (можно поставить галочку «Создать ярлык на рабочем столе»).
- После установки тренажёр появится в меню «Пуск» и/или на рабочем столе, запуск — как обычной программы.

## Запуск в Docker (опционально)

> Важно: это GUI‑приложение на `tkinter`, внутри Docker его имеет смысл запускать либо для сборки/тестов, либо с прокинутым X‑сервером/VNC. Для обычных пользователей проще раздавать `TypingTrainer.exe` или установщик.

1. Собрать образ:

```bash
cd typing_trainer
docker build -t typing-trainer .
```

2. Запуск (Linux + X11, пример):

```bash
docker run --rm -e DISPLAY=$DISPLAY \
    -v /tmp/.X11-unix:/tmp/.X11-unix \
    typing-trainer
```

На Windows обычно достаточно использовать собранный `TypingTrainer.exe` или установщик, а Docker держать для разработки.
